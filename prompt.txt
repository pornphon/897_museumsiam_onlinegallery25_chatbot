ทำระบบ RAG โดยใช้ CSV นี้ เพื่อตอบคำถามวัตถุโบราณ
จัดทำ Tools,Agent, Router และเชื่อม Agent เข้ากับTool เพื่อให้ agent เรียกใช้ โดย Tools จะมี Tools ดังนี้
-ค้นหาจากชื่อวัตถุโบราณ(name_th) โดยใช้ vector search แบบ similarity โดยจะได้ผลลัพธ์เป็น ID ของวัตถุโบราณกลับมา
-ค้นหาจากคุณสมบัติวัตถุโบราณ(material_tags) โดยใช้ vector search แบบ similarity โดยจะได้ผลลัพธ์เป็น ID ของวัตถุโบราณกลับมา
-ค้นหาจากคำอธิบายวัตถุโบราณ(artistic_description_th) โดยใช้ vector search แบบ similarityโดยจะได้ผลลัพธ์เป็น ID ของวัตถุโบราณกลับมา
-ค้นหาจากสถานที่ผลิตวัตถุโบราณโดยจะได้ผลลัพธ์เป็น ID ของวัตถุโบราณกลับมา
-ค้นหาจากคำอธิบายของวัตถุโบราณโดยจะได้ผลลัพธ์เป็น ID ของวัตถุโบราณกลับมา
